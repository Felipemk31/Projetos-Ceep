atributo type na lista ordenada, é utilizado para definir qual o sistema de organização entre os disponíveis deverá ser aplicado aos itens da lista.
as formas possíveis são:
type ="1" algarismo do sistema numérico cardinal(1,2,3 etc...)
type ="i" algarismos em minúsculo do sistema numérico romano(i,ii,iii,iv etc...)
type ="I" algarismos em maiúsculo do sistema numérico romano(I,II,III,IV etc...)
type ="a" caracteres em minúsculo do sistema alfabético (a,b,c, etc...)
type ="A" caracteres em maiúsculo do sistema alfabético (A,B,C, etc...)

o atributo type pode ser usado além da tag ol, a tag ul e dentro do atributo li(personalizando apenas os itens desejados).
O objetivo do atributo type é realizar a alteração do tipo de marcador. O atributo type é ultilizado para definir qual simbolo gráfico, entre os 3 tipos disponiveis para a lista não ordenada.

  • Adilson
  • Adrian
  • Amanda
  • Ana
