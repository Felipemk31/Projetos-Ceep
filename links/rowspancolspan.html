atributo colspan

O atributo "colspan"(O "col" vem de column (coluna) e span quer dizer expandir, abranger).
Serve para expandir uma coluna para abranger varias celulas(em uma coluna) da tabela.

exemplo colspan

titulo da tabela
linha 1 coluna 1 linha 1 coluna 2 linha 1 coluna 3
linha 2 coluna 1 linha 2 coluna 2 linha 2 coluna 3
linha 3 coluna 1 linha 3 coluna 2 linha 3 coluna 3

atributo rowspan

O atributo "rowspan"(O "row" vem de row (linha) e span quer dizer expandir, abranger).
Serve para expandir uma linha para abranger varias celulas(em uma linha) da tabela.

exemplo rowspan
